# Online Quiz and Examination System

## Features
- User Login System
- Quiz with Timer
- Score Calculation
- Result Analysis with Graphs
- File Handling (No Database)

## Technologies Used
- Java
- Swing (GUI)
- File Handling

## How to Run
javac -d . src/*.java
java Main
